# Image Enhancer - 正體中文說明文件

## Skill 概述

### 功能
Image Enhancer skill 提升圖片品質，特別是螢幕截圖，透過增強解析度、銳利度和清晰度，讓圖片看起來更專業。非常適合為簡報、文件或社群媒體貼文準備圖片。

### 解決的問題
內容創作者和專業人士在處理圖片時常面臨以下挑戰：

1. **截圖品質不佳**：螢幕截圖在放大或印刷時顯得模糊
2. **解析度不足**：低解析度圖片無法滿足高品質輸出需求
3. **壓縮偽影**：過度壓縮導致圖片出現雜訊和失真
4. **模糊照片**：相機晃動或對焦不準導致影像不清晰
5. **社群媒體需求**：不同平台需要不同的圖片尺寸和品質
6. **批次處理耗時**：需要處理多張圖片時手動操作費時

這個 skill 自動化圖片增強過程，使用智慧演算法提升品質，同時保留原始檔案作為備份。

## 核心功能分析

### 1. 圖片品質分析

**檢查項目**：
- 當前解析度
- 圖片格式（PNG、JPG、WebP 等）
- 壓縮品質
- 銳利度和清晰度
- 是否有壓縮偽影或雜訊

**分析輸出**：
```
分析 screenshot-2024.png...

當前規格：
- 解析度：1920x1080
- 格式：PNG
- 品質：良好，但稍微模糊
```

### 2. 解析度增強

**提升技術**：
- 智慧放大演算法
- 邊緣保留
- 細節增強
- 適應性銳化

**常見提升目標**：
- 標準 HD (1920x1080) → Retina (2560x1440)
- 2K → 4K
- 為特定用途（印刷、簡報）最佳化

### 3. 銳利度改善

**增強方式**：
- 邊緣檢測和強化
- 文字清晰度提升（特別重要對截圖）
- 適應性去模糊
- 保持自然外觀

### 4. 偽影減少

**清理項目**：
- JPEG 壓縮塊狀偽影
- 色彩帶狀（color banding）
- 雜訊和顆粒
- 模糊邊緣

### 5. 用途最佳化

**根據目標調整**：
- **網頁**：平衡品質和檔案大小
- **印刷**：最高解析度和色彩準確度
- **社群媒體**：符合平台規格
- **簡報**：為大螢幕最佳化

### 6. 批次處理

**支援功能**：
- 處理整個資料夾的圖片
- 套用一致的增強設定
- 自動命名和組織
- 保留原始檔案

## 最佳實踐

### 使用技巧

1. **指定輸出格式**：如果有特定需求（PNG 保持品質，JPG 縮小檔案）
2. **說明用途**：提及圖片的最終用途（網頁、印刷、社群媒體）
3. **批次處理**：一次處理整個資料夾節省時間
4. **保留原始檔**：Claude 會自動保留，但可以明確要求
5. **指定品質目標**：如「適合印刷」或「網頁最佳化」

### 圖片品質建議

#### 解析度目標

**網頁使用**：
- 標準：1920x1080
- Retina：2560x1440
- 4K：3840x2160

**印刷使用**：
- 最少 300 DPI
- 較大尺寸需更高解析度

**社群媒體**：
- Twitter：1200x675
- Facebook：1200x630
- Instagram：1080x1080（正方形）或 1080x1350（直式）
- LinkedIn：1200x627

#### 檔案格式選擇

**PNG**：
- 需要透明背景
- 高品質要求
- 截圖和圖表
- 較大檔案

**JPG**：
- 照片
- 無需透明背景
- 較小檔案大小
- 輕微品質損失可接受

**WebP**：
- 現代網頁
- 更好的壓縮
- 品質和大小平衡
- 較新的格式

### 增強限制

**了解限制**：
- 無法從極低品質中「創造」細節
- 嚴重模糊或損壞的圖片改善有限
- 放大有實際限制（通常 2-4 倍效果最佳）
- AI 放大可能引入人工痕跡
