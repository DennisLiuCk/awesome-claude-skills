# Video Downloader - 正體中文說明文件

## Skill 概述

### 功能
Video Downloader skill 從 YouTube 和其他平台下載影片，供離線觀看、編輯或存檔使用。處理各種格式和品質選項，讓您輕鬆保存線上影片內容。

### 解決的問題
內容創作者和學習者在保存線上影片時常面臨以下挑戰：

1. **需要離線存取**：沒有網路連線時無法觀看
2. **內容可能消失**：影片可能被刪除或下架
3. **編輯需求**：需要影片檔案進行後製或重新利用
4. **學習和參考**：想保存教學內容供日後參考
5. **品質選擇困難**：不確定應該下載哪種解析度
6. **格式不相容**：需要特定格式供特定用途

這個 skill 自動化影片下載過程，支援多種平台、格式和品質選項。

## 核心功能分析

### 1. 影片下載

**支援平台**：
- YouTube（主要支援）
- 其他影片平台（視工具支援度）

**下載功能**：
- 獲取各種解析度的影片
- 下載影片和音訊
- 保留原始品質
- 快速可靠的下載

### 2. 品質選擇

**解析度選項**：
- **480p**：標準畫質，檔案較小
- **720p**：HD 畫質，平衡品質和大小
- **1080p**：Full HD，高品質
- **4K**：超高清（如可用）

**品質考量**：
- 更高解析度 = 更大檔案
- 選擇適合用途的品質
- 考慮儲存空間限制

### 3. 格式選項

**影片格式**：
- **MP4**：最通用，廣泛支援
- **WebM**：網頁優化格式
- 其他格式（視工具支援）

**音訊格式**：
- **MP3**：音訊提取
- 僅音訊下載（用於 podcast、音樂）

### 4. 批次下載

**支援功能**：
- 下載多個影片
- 播放清單下載
- 批次處理多個 URL
- 自動命名和組織

### 5. 元數據保存

**保存資訊**：
- 影片標題
- 描述
- 縮圖圖片
- 頻道資訊
- 上傳日期

## Claude Code 使用方式

### 觸發時機

Claude Code 會在以下情況使用此 Skill：

1. **基本下載**
   ```
   下載這個 YouTube 影片：https://youtube.com/watch?v=...
   ```

2. **指定品質**
   ```
   以 1080p 品質下載這個影片
   ```

3. **僅音訊**
   ```
   從這個 YouTube 影片下載音訊為 MP3
   ```

4. **播放清單**
   ```
   下載這個 YouTube 播放清單的所有影片：[URL]
   ```

5. **批次下載**
   ```
   下載這 5 個 YouTube 影片：
   1. [URL]
   2. [URL]
   ...
   ```

## 最佳實踐

### 使用技巧

1. **指定品質**：如果有檔案大小考量（720p vs 1080p）
2. **僅音訊用於 podcast**：音樂或 podcast 使用僅音訊節省空間
3. **下載到專用資料夾**：保持組織性
4. **慢連線前檢查檔案大小**：避免長時間下載
5. **使用播放清單功能**：批次下載相關影片

### 品質建議

**用途匹配品質**：
- **背景播放**：480p 或僅音訊
- **一般觀看**：720p（HD）
- **高品質觀看**：1080p
- **專業用途**：4K（如可用且需要）

### 儲存管理

**檔案大小參考**：
- 480p：~100-200 MB/小時
- 720p：~200-500 MB/小時
- 1080p：~400-800 MB/小時
- 4K：~2-4 GB/小時
