# Brand Guidelines - 正體中文說明文件

## Skill 概述

### 功能
Brand Guidelines skill 為任何可能受益於 Anthropic 外觀和風格的 artifact 應用官方品牌色彩和排版。當需要品牌色彩、樣式指南、視覺格式化或公司設計標準時使用。

### 解決的問題
在建立與 Anthropic 相關的文件、簡報或其他視覺內容時，設計者常面臨以下挑戰：
1. **品牌一致性**：難以記住和正確應用官方品牌色彩
2. **字型選擇**：不確定使用哪些字型符合品牌指南
3. **視覺風格**：缺乏清晰的品牌視覺標準參考
4. **重複查找**：每次都需要重新查找品牌資訊

這個 skill 將 Anthropic 的品牌指南整合到 Claude 的知識中，確保產出的內容符合品牌標準。

## 核心內容分析

### 品牌色彩系統

#### 主要色彩
- **Dark** (`#141413`): 主要文字和深色背景
- **Light** (`#faf9f5`): 淺色背景和深色背景上的文字
- **Mid Gray** (`#b0aea5`): 次要元素
- **Light Gray** (`#e8e6dc`): 微妙的背景

#### 強調色彩
- **Orange** (`#d97757`): 主要強調色
- **Blue** (`#6a9bcc`): 次要強調色
- **Green** (`#788c5d`): 第三強調色

### 字型系統

- **標題字型**: Poppins（備用字型：Arial）
- **內文字型**: Lora（備用字型：Georgia）
- **字型回退機制**: 當自訂字型不可用時自動使用系統字型

### 設計策略

#### 1. 智慧字型應用
- 自動為 24pt 及以上的標題應用 Poppins 字型
- 為內文應用 Lora 字型
- 在字型不可用時自動回退到 Arial/Georgia
- 保持所有系統的可讀性

#### 2. 文字樣式
- 根據背景智慧選擇文字顏色
- 保持文字層次結構
- 維護原有格式

#### 3. 形狀和強調色
- 非文字形狀使用強調色
- 循環使用橙色、藍色和綠色強調色
- 在保持品牌一致性的同時維持視覺趣味

## 技術細節

### 色彩應用
- 使用 RGB 色彩值確保精確的品牌匹配
- 透過 python-pptx 的 RGBColor 類別應用
- 在不同系統間維持色彩保真度

### 字型管理
- 優先使用系統安裝的 Poppins 和 Lora 字型
- 提供自動回退到 Arial（標題）和 Georgia（內文）
- 無需安裝字型即可工作 - 使用現有系統字型
- 為了最佳效果，建議在環境中預先安裝 Poppins 和 Lora 字型

## 最佳實踐

### 使用建議
1. **預先安裝字型**: 為了最佳效果，在系統中安裝 Poppins 和 Lora 字型
2. **指定品牌需求**: 明確告訴 Claude 需要應用 Anthropic 品牌樣式
3. **檢查輸出**: 確認色彩和字型正確應用
4. **一致性**: 在所有相關文件中使用相同的品牌標準

### 注意事項
- 字型回退機制確保即使沒有自訂字型也能正常工作
- 色彩值是固定的，確保品牌一致性
- 強調色會循環使用以保持視覺趣味
