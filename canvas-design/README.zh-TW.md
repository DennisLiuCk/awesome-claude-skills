# Canvas Design - 正體中文說明文件

## Skill 概述

### 功能
Canvas Design skill 是一個視覺藝術創作工具，透過設計哲學創建美麗的視覺藝術作品。它能夠生成海報、藝術作品、設計圖以及其他靜態視覺作品，輸出格式包括 PNG 和 PDF 文件。這個 skill 強調純粹的視覺表達，而非文字說明。

### 解決的問題
設計師和創作者在創作視覺作品時常面臨以下挑戰：

1. **缺乏設計理念**：不知從何開始構思設計方向
2. **視覺表達困難**：難以將抽象概念轉化為視覺語言
3. **專業度不足**：作品看起來不夠精緻或專業
4. **設計一致性**：難以維持整體設計風格的統一性
5. **創意瓶頸**：缺乏靈感和創新的設計方法

這個 skill 透過兩步驟流程（設計哲學創建 + 視覺表達）來解決這些問題，確保每個作品都有深厚的概念基礎和精湛的執行品質。

## 核心功能分析

### 1. 設計哲學創建

#### 哲學命名
為設計運動命名（1-2個詞），例如：
- "Brutalist Joy" （野獸派喜悅）
- "Chromatic Silence" （色彩寂靜）
- "Metabolist Dreams" （代謝主義夢境）

#### 哲學闡述
透過 4-6 段文字詳細描述設計哲學，涵蓋：
- 空間與形式
- 色彩與材質
- 規模與節奏
- 構圖與平衡
- 視覺層次

#### 設計範例
SKILL.md 提供了多個設計哲學範例：

**「Concrete Poetry」（具象詩歌）**
- 理念：透過紀念性形式和大膽幾何表達溝通
- 視覺表現：巨大色塊、雕塑式排版、波蘭海報能量結合柯比意風格

**「Chromatic Language」（色彩語言）**
- 理念：色彩作為主要資訊系統
- 視覺表現：幾何精度、色彩區域創造意義

**「Analog Meditation」（類比冥想）**
- 理念：透過質感和留白進行安靜的視覺沉思
- 視覺表現：紙張紋理、墨水渲染、大量負空間

### 2. 畫布創作

#### 視覺優先
- 90% 視覺設計
- 10% 必要文字
- 文字作為視覺元素整合

#### 精湛工藝
作品必須呈現：
- 看起來花費無數小時精心製作
- 來自頂尖專業人士的作品
- 每個細節都經過細心雕琢
- 可在博物館或畫廊展示的品質

#### 概念編織
- 將主題作為微妙的概念線索編織進設計
- 精緻而不張揚的引用
- 熟悉主題的人能直覺感受到
- 其他人則單純欣賞精湛的抽象構圖

### 3. 多頁選項

當需要多頁作品時：
- 沿用相同設計哲學
- 每頁獨特但相關
- 如同咖啡桌書的多個頁面
- 以精緻方式講述視覺故事

## 設計策略

### 1. 兩步驟創作流程

**步驟一：設計哲學創建**
- 輸出 .md 檔案
- 定義美學世界觀
- 提供創意詮釋空間

**步驟二：視覺表達**
- 輸出 .pdf 或 .png 檔案
- 在畫布上表達哲學
- 實現精湛工藝

### 2. 哲學驅動設計

設計哲學作為：
- 美學基礎
- 創意指南
- 品質標準
- 概念框架

### 3. 專業級執行

#### 排版策略
- 文字極簡但視覺優先
- 根據情境調整文字規模
- 無重疊、無溢出
- 使用 `./canvas-fonts` 目錄中的字型
- 文字本身成為藝術的一部分

#### 視覺品質
- 重複圖案和完美形狀
- 系統化的視覺語言
- 臨床精度的排版
- 有限但協調的色彩調色盤

### 4. 反覆精煉

**最終步驟的關鍵原則**：
- 避免添加更多圖形
- 精煉現有元素
- 尊重設計哲學和極簡主義
- 使作品更具凝聚力
- 追求博物館級品質

1. **創作視覺藝術作品時**
   - 範例：「為我設計一張海報」

2. **需要專業設計時**

   - 範例：「設計一張活動海報」
   - 範例：「創建一個藝術裝置的視覺稿」

### 典型使用場景

```
為一個關於永續發展的活動設計海報
```

1. 創建「Organic Systems」或類似設計哲學
2. 定義自然、成長、系統的視覺語言
3. 在畫布上表達：圓潤形式、有機排列、自然色彩
4. 微妙地編織永續發展的概念
5. 輸出精緻的 PDF 或 PNG

#### 場景 2：抽象藝術創作

```
創建一個關於科技與人性的抽象藝術作品
```

Claude 會：
1. 構思「Geometric Silence」或「Chromatic Language」哲學
2. 平衡技術精度與人性溫度
3. 使用幾何形式和色彩表達張力
4. 創建看似花費數日精心製作的作品

#### 場景 3：多頁設計系列

```
為咖啡品牌設計一系列 5 頁的視覺作品
```

Claude 會：
1. 創建統一的設計哲學（如「Analog Meditation」）
2. 第一頁建立視覺基調
3. 後續頁面延續風格但各具特色
4. 以精緻方式講述咖啡文化的視覺故事

## 使用的 Prompt 策略

### 1. 分階段創作指令

**第一階段：哲學創建**
- 明確分離概念構思與視覺執行
- 要求創建美學運動宣言
- 提供多個設計哲學範例
- 強調為下一階段留下創意詮釋空間

**第二階段：視覺表達**
- 基於已建立的哲學
- 強調視覺優先、文字最少
- 反覆強調精湛工藝的重要性

### 2. 重複強調品質

在 SKILL.md 中多次強調：
- "meticulously crafted"（細心製作）
- "painstaking attention"（細緻入微的關注）
- "master-level execution"（大師級執行）
- "countless hours"（無數小時）
- "top of their field"（該領域頂尖）

### 3. 具體的設計範例

提供 5 個完整的設計哲學範例：
- Concrete Poetry
- Chromatic Language
- Analog Meditation
- Organic Systems
- Geometric Silence

每個範例包含理念和視覺表達方式。

### 4. 明確的約束條件

**必須遵守**：
- 文字稀疏、僅必要內容
- 空間表達優先於文字說明
- 無重疊、無溢出
- 每個元素都有呼吸空間
- 不同字型的使用

**避免**：
- 重複相同設計要點
- 卡通化或業餘風格
- 過度解釋性的文字
- 添加更多圖形而非精煉現有元素

### 5. 概念編織技巧

**關鍵原則**：
- 主題作為微妙的概念線索
- 不是文字層面的引用
- 精緻的引用增強作品深度
- 如同爵士樂手引用另一首歌

## 使用的工具（Tools）

此 Skill 主要依賴以下 Claude Code 工具：

### 1. **Write Tool**
- 建立設計哲學 .md 檔案
- 生成設計理念文件

### 2. **Canvas/Drawing Libraries**
- 創建 PDF 和 PNG 檔案
- 繪製幾何形狀和圖案
- 應用色彩和視覺效果

### 3. **Font Management**
- 存取 `./canvas-fonts` 目錄
- 下載和使用所需字型
- 整合排版到視覺作品

### 4. **Python Graphics Libraries**
可能使用的庫：
- **Pillow/PIL**: 圖像處理和創建
- **Cairo**: 向量圖形繪製
- **ReportLab**: PDF 生成
- **Matplotlib**: 視覺化和圖形創建

## 最佳實踐

### 使用建議

1. **提供清晰的概念**：說明你想表達的核心理念或主題
2. **信任創作流程**：讓 Claude 先創建哲學，再表達視覺
3. **要求精煉**：如果初稿不夠完美，要求進一步精煉而非重做
4. **指定頁數**：明確說明需要單頁還是多頁設計
5. **描述情境**：說明作品的用途（海報、藝術品、活動視覺等）

### 品質標準

#### 視覺品質
- 看起來像花費數日精心製作
- 可以展示作為專業作品集
- 符合博物館或畫廊展示標準
- 每個細節都經過深思熟慮

#### 概念深度
- 有明確的設計哲學支撐
- 微妙的概念編織
- 視覺語言一致
- 空間和形式有意義

#### 技術執行
- 無格式錯誤
- 無重疊或溢出
- 完美的對齊和間距
- 高解析度輸出

### 注意事項

1. **避免卡通風格**：即使是輕鬆主題，仍保持精緻度
2. **文字極簡**：除非特定需求，否則文字應該最少
3. **原創性**：不複製現有藝術家作品，避免版權問題
4. **概念微妙**：引用應該精緻而不張揚
5. **反覆精煉**：最後階段專注於精煉而非添加

## 總結

### 成功要素

1. **兩步驟流程**：分離哲學構思與視覺執行
2. **視覺優先**：90% 視覺、10% 文字
3. **精湛工藝**：反覆強調大師級品質
4. **概念深度**：微妙的主題編織
5. **反覆精煉**：持續改進直到博物館級品質

### 設計哲學類型

- **Concrete Poetry**: 紀念性形式、大膽幾何
- **Chromatic Language**: 色彩作為資訊系統
- **Analog Meditation**: 質感、留白、沉思
- **Organic Systems**: 自然聚集、模組化成長
- **Geometric Silence**: 純粹秩序、克制美學

### 適用場景

- 海報設計
- 藝術作品創作
- 活動視覺設計
- 品牌視覺表達
- 概念藝術
- 靜態設計作品
- 視覺敘事系列

### 核心理念

這不僅是製作帶有裝飾的文件，而是創作**藝術品**。每個作品都應該：
- 有深厚的概念基礎
- 展現精湛的視覺工藝
- 透過形式、色彩、空間溝通
- 達到專業展示的品質標準
