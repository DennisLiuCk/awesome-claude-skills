# Raffle Winner Picker - 正體中文說明文件

## Skill 概述

### 功能
Raffle Winner Picker skill 從清單、試算表或 Google Sheets 中隨機挑選贈品、抽獎和競賽的贏家。確保公平、無偏見的選擇，並提供透明度。

### 解決的問題
活動組織者和行銷人員在抽選贏家時常面臨以下挑戰：

1. **公平性疑慮**：手動選擇可能被質疑有偏見
2. **透明度需求**：參與者想知道選擇過程是公正的
3. **重複問題**：需要確保同一人不會贏兩次
4. **時間消耗**：手動從數百個參賽者中挑選很費時
5. **記錄保存**：需要記錄選擇過程以備查證
6. **多贏家選擇**：需要一次選出多個贏家和候補

這個 skill 提供可驗證的隨機選擇過程，確保公平性和透明度。

## 核心功能分析

### 1. 隨機選擇

**加密安全**：
- 使用加密隨機數生成
- 無偏見或模式
- 透明的過程
- 可用種子值重複驗證（供第三方驗證）

### 2. 多種來源支援

**支援格式**：
- **Google Sheets**：直接從雲端試算表選擇
- **CSV 檔案**：從本地試算表
- **Excel (XLSX)**：支援 Excel 格式
- **純文字清單**：簡單的名單

### 3. 多贏家選擇

**功能**：
- 選擇一個或多個贏家
- 防止重複（同一人不會贏兩次）
- 選擇主要贏家和候補
- 分階段抽獎（先選第一名，再選第二名等）

### 4. 透明結果

**輸出資訊**：
- 贏家詳細資訊（姓名、電子郵件等）
- 選擇方法說明
- 時間戳記（UTC）
- 總參賽者數
- 隨機種子（供驗證）

### 5. 贏家詳情

**顯示資訊**：
- 行號（從資料來源）
- 所有相關欄位（姓名、電子郵件、備註等）
- 參賽日期（如可用）
- 其他自訂欄位

### 6. 排除功能

**支援排除**：
- 排除先前贏家
- 排除特定人員或條目
- 排除不合格的參賽者
- 基於條件篩選

## Claude Code 使用方式

### 觸發時機

Claude Code 會在以下情況使用此 Skill：

1. **從 Google Sheets 選擇**
   ```
   從這個 Google Sheet 隨機挑選一行以選擇贈品贏家：[Sheet URL]
   ```

2. **從本地檔案選擇**
   ```
   從 entries.csv 隨機挑選 3 個贏家
   ```

3. **從清單選擇**
   ```
   從這個清單隨機挑選一個贏家：
   - Alice (alice@email.com)
   - Bob (bob@email.com)
   - Carol (carol@email.com)
   ...
   ```

4. **多贏家選擇**
   ```
   從 contest-entries.xlsx 隨機挑選 5 個贏家，確保無重複
   ```

### 典型使用場景

#### 場景 1：社群媒體贈品（受 Lenny 的使用啟發）

**使用者**：「從這個 Google Sheet 隨機挑選一行以選擇贈品贏家。」

**輸出**：
```

```
從 attendees.csv 隨機挑選 10 個贏家

**Claude 會**：
1. 載入 CSV 檔案
2. 隨機選擇 10 個不重複的參賽者
3. 顯示所有贏家詳情
4. 提供匯出清單選項

#### 場景 3：排除先前贏家

```

```

**Claude 會**：
1. 載入參賽者清單
2. 過濾掉指定的人
3. 從剩餘參賽者中隨機選擇
4. 確認排除生效

## 使用的工具（Tools）

此 Skill 主要依賴以下 Claude Code 工具：

### 1. **WebFetch Tool**
- 存取 Google Sheets
- 讀取雲端試算表
- 獲取參賽者數據

### 2. **Read Tool**
- 讀取本地 CSV 檔案
- 讀取 Excel 檔案
- 處理文字清單

- **secrets** 模組：加密安全隨機數

- 防止可預測的模式

### 4. **Write Tool**
- 匯出贏家清單
- 保存結果記錄
- 創建驗證文件
