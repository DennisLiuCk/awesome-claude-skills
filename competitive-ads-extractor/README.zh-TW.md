# Competitive Ads Extractor - 正體中文說明文件

## Skill 概述

### 功能
Competitive Ads Extractor 從廣告庫（Facebook、LinkedIn 等）中提取和分析競爭對手的廣告，以了解哪些訊息、問題和創意方法有效。幫助啟發和改進您自己的廣告活動。

### 解決的問題
行銷人員在規劃廣告活動時面臨以下挑戰：

1. **缺乏競爭洞察**：不知道競爭對手正在使用什麼訊息策略
2. **創意靈感不足**：難以找到有效的廣告創意模式
3. **手動分析耗時**：逐一查看和分析競爭對手廣告非常費時
4. **訊息定位不明**：不確定哪些痛點和價值主張最能引起共鳴
5. **缺乏系統性**：沒有結構化的方法來追蹤和比較廣告趨勢

這個 skill 自動化了競爭廣告的提取、截圖保存、訊息分析和模式識別過程，將數天的手動工作縮短為幾分鐘。

## 功能分析

### 1. 廣告提取
- 從 Facebook Ad Library、LinkedIn 等平台抓取廣告
- 支援指定公司或競爭對手清單
- 可按時間範圍篩選（當前活躍、特定期間等）

### 2. 視覺保存
- 自動截取所有廣告的圖像
- 組織保存到結構化目錄
- 便於後續參考和比較

### 3. 訊息分析
識別和分類：
- 強調的問題和痛點
- 目標使用場景
- 價值主張
- 受眾定位

### 4. 廣告分類
根據以下維度分組：
- 主題（生產力、協作、效率等）
- 受眾（新創、企業、團隊、個人）
- 格式（靜態圖片、影片、輪播等）

### 5. 模式識別
發現：
- 成功的創意模式（前後對比、功能展示等）
- 有效的文案公式
- 視覺風格趨勢
- CTA（行動呼籲）模式

### 6. 洞察提供
說明：
- 為什麼某些廣告可能表現良好
- 哪些訊息引起共鳴
- 競爭對手的定位策略
- 可應用於自己廣告的建議

## 設計策略

### 1. 全面性分析
不僅提取廣告，更提供深度分析：
- 量化數據（廣告數量、主題分佈）
- 質化洞察（文案模式、視覺風格）
- 可操作建議（如何應用這些洞察）

### 2. 結構化輸出
提供多種格式的輸出：
- **截圖**: 所有廣告保存為圖像
- **分析報告**: Markdown 格式的洞察摘要
- **試算表**: CSV 格式的廣告文案、CTA、主題
- **簡報**: 頂級表現者的視覺集合
- **模式庫**: 按方法分類

### 3. 倫理和法律考量
明確說明使用界限：
- ✓ 僅用於研究和靈感
- ✓ 不直接複製廣告
- ✓ 尊重智慧財產權
- ✓ 使用洞察創作原創內容
- ✗ 不抄襲文案或竊取設計

## Claude Code 使用方式

### 觸發時機
Claude Code 會在以下情況使用此 Skill：

1. **競爭研究需求**
   ```
   提取 [競爭對手名稱] 在 Facebook Ad Library 上的所有當前廣告
   ```

2. **訊息分析**
   ```
   從 [公司] 抓取廣告並分析他們的訊息策略
   ```

3. **多競爭對手比較**
   ```
   提取這 5 個競爭對手的廣告：[清單]。比較他們的方法並告訴我什麼有效。
   ```

4. **特定平台分析**
   ```
   獲取 [競爭對手] 的 LinkedIn 廣告並分析他們的 B2B 定位策略
   ```

5. **趨勢追蹤**
   ```
   比較 [競爭對手] Q1 和 Q2 的廣告。訊息有什麼變化？
   ```

### 典型工作流程

#### 廣告活動規劃
1. 提取競爭對手廣告
2. 識別成功模式
3. 注意他們訊息中的空白
4. 腦力激盪獨特角度
5. 草擬測試廣告變體

#### 定位研究
1. 獲取 5 個競爭對手的廣告
2. 繪製他們的定位
3. 找出服務不足的角度
4. 開發差異化訊息
5. 針對他們的方法進行測試

#### 創意靈感
1. 按主題提取廣告
2. 分析視覺模式
3. 注意色彩和版面趨勢
4. 改編成功模式
5. 創建原創變體

## 最佳實踐

### 研究技巧
1. **定期監控**: 每月檢查變化
2. **廣泛研究**: 也查看相鄰競爭對手
3. **保存所有內容**: 建立參考庫
4. **測試洞察**: 進行自己的實驗
5. **追蹤表現**: A/B 測試受啟發的概念
6. **保持原創**: 用於靈感，不是複製
7. **多平台**: 比較 Facebook、LinkedIn、TikTok 等

### 分析技巧
1. **尋找模式**: 哪些主題重複？
2. **隨時間追蹤**: 每月保存廣告以查看演變
3. **測試假設**: 為您的品牌調整成功模式
4. **按受眾細分**: 不同目標的不同訊息
5. **比較平台**: LinkedIn vs Facebook 訊息不同
