# Domain Name Brainstormer - 正體中文說明文件

## Skill 概述

### 功能
Domain Name Brainstormer skill 為您的專案生成創意域名建議，並檢查多個頂級域名（.com、.io、.dev、.ai 等）的可用性。這個工具能在幾分鐘內完成原本需要數小時的腦力激盪和手動檢查工作。

### 解決的問題
創業者、開發者和企業在選擇域名時常面臨以下挑戰：

1. **創意枯竭**：難以想出有意義且吸引人的域名
2. **可用性檢查耗時**：需要逐一檢查每個想到的域名是否已被註冊
3. **首選被佔用**：理想的域名通常已被使用
4. **多個 TLD 比較**：不確定應該選擇 .com、.io 還是其他後綴
5. **品牌適配度**：難以評估域名是否符合品牌定位
6. **價格考量**：不知道某些域名是否為高價的優質域名

這個 skill 自動化了域名腦力激盪和可用性檢查過程，提供具體建議和替代方案。

## 核心功能分析

### 1. 專案理解

**分析面向**：
- 產品或服務的核心功能
- 目標受眾
- 產業和市場定位
- 獨特價值主張
- 品牌個性和語調

**資訊來源**：
- 用戶描述
- 程式碼庫分析（如果在專案目錄中運行）
- 功能特性清單
- 目標市場說明

### 2. 創意域名生成

**命名策略**：

**描述性名稱**：
- 直接說明產品功能
- 範例：CodeShare、SnippetBox、DevPaste

**複合詞**：
- 結合相關詞彙
- 範例：PixelStudio、CodeClip、DataFlow

**簡短縮寫**：
- 簡潔易記
- 範例：8 個字元以內

**創意變化**：
- 拼字變化
- 詞尾變化
- 前綴/後綴組合

### 3. 多 TLD 可用性檢查

檢查多個頂級域名：
- **.com**：最通用和可信賴
- **.io**：科技新創公司流行
- **.dev**：開發者工具
- **.ai**：人工智慧產品
- **.app**：應用程式
- **.co**：.com 的替代選項
- **.xyz**：現代創意專案
- **.design**：創意/設計機構
- **.tech**：科技公司

**檢查結果**：
- ✓ 可用
- ✗ 已被佔用
- $ 高價優質域名
- 預估價格範圍

### 4. 品牌洞察

**為每個建議提供**：
- **為什麼有效**：說明命名理由
- **品牌適配度**：評估與專案的契合度
- **記憶度評分**：易記程度
- **發音簡易度**：口語傳播友善度
- **獨特性**：是否足夠突出

### 5. 優先級排序

**評估標準**：
- 可用性
- 長度（越短越好）
- 清晰度
- 品牌潛力
- TLD 適合度
- 價格合理性

**排名輸出**：
- 🏆 首選推薦
- 🥈 次選方案
- 替代選項

### 6. 替代方案

當首選不可用時：
- 提供變化版本
- 建議不同 TLD
- 相關詞彙替換
- 創意拼字變化

## 最佳實踐

### 使用技巧

1. **提供清晰的專案描述**：越詳細越好
2. **說明目標受眾**：幫助生成適合的名稱
3. **指定偏好**：長度、風格、TLD 偏好
4. **從專案目錄運行**：讓 Claude 分析程式碼以更好理解
5. **要求多輪迭代**：第一批不滿意就要求更多選項

### 域名選擇建議

#### 評估標準

**簡短度**：
- 5-10 個字元：優秀
- 11-15 個字元：良好
- 16+ 個字元：考慮縮短

**清晰度**：
- 一聽就懂
- 拼寫不會錯
- 無歧義發音

**品牌潛力**：
- 獨特但不怪異
- 可以建立故事
- 視覺化友善（logo 設計）

**SEO 考量**：
- 包含相關關鍵字（但不過度）
- 避免數字和連字號
- 選擇適合的 TLD

### 行動建議

1. **快速行動**：好域名很快被註冊
2. **註冊多個變化版本**：同時拿下 .com 和 .io 以保護品牌
3. **避免數字**：口語溝通困難
4. **檢查社群媒體**：確保 @用戶名 也可用
5. **大聲說出來**：測試是否易於發音
6. **檢查商標**：確保無法律衝突
7. **長期思考**：5 年後仍然適用嗎？

### 價格參考

**標準域名**：~$10-15/年
**優質 TLD**（.io、.ai）：~$30-50/年
**已被佔用域名**：市場價格（如有列出）
**優質域名**：數百至數千美元
